# Udacity Project 5


### Prerequisites
```
Truffle v5.0.24 (core: 5.0.24)
Solidity v0.5.0 (solc-js)
Node v10.16.0
Web3.js v1.0.0-beta.37
```

### compile and test
```
truffle develop
compile
test
```

### deploy contract
```
truffle migrate --reset --network rinkeby            
```

### run
```
cd app
npm run dev
```
